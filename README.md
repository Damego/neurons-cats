# Классификация породы котов на датасете “pop-cats”


## Информация

Данный проект реализует пайплайн дообучения нейросети ResNet18 для классификации изображений на датасете из ~10 000 изображений кошек. 
В процессе разработки были решены задачи обработки данных, нормализации и аугментации.

#### Примеры кошек

![cats](https://github.com/Damego/neurons-cats/blob/main/cats_example.png?raw=true)

### 🏆 Результаты
- **Точность на валидации:** ~89-90%.
- **Базовая модель:** ResNet18.
- **Оптимизатор:** AdamW с Weight Decay.
- **Архитектура:** Linear -> ReLU -> Dropout(0.5) -> Linear

С использованем batch size = 64, lr = 3e-5

#### График

![график](https://github.com/Damego/neurons-cats/blob/main/%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D0%BA.png?raw=true)

Точность для тестовой выборки перестала расти с 10 эпохи, остановившись на отметке в ~0.895 и совершала колебания от 0.89 до 0.9 на оставшихся эпохах. Функция потерь вела себя похоже.
Точность обучающей выборки росла с каждой эпохи и достигла 0.995 к концу обучения.

## 📊 Методы и решения

### 🎨 Аугментация данных
Для увеличения размера выборки и улучшения обобщения использовались

- `RandomResizedCrop`: масштабирование и обрезка.
- `ColorJitter`: изменение яркости и контраста.
- `RandomHorizontalFlip`: зеркальное отражение.

## Early Stopping

Автоматическая остановка обучения при отсутствии улучшения метрик на валидации в течение patience эпох. Предотвращает переобучение модели и сохраняет веса лучшей эпохи.

### Что пробовали

- Увеличение batch size и lr приводили к сильному ухудшению точности на валидации, ~0.03-0.04 на первых эпохах.
- Смена базовой модели ResNet18 на EfficientNet B0 и B3, но длительность обучения одной эпохи была ~12 и ~24 минут соответственно, при этом результат не отличался от ResNet18. Пробовалось, чтобы повысить точность больше, чем 0.9 при ResNet18.
